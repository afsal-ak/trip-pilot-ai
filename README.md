# ✈️ Trip Pilot AI

An AI-powered travel itinerary generation platform built with the **MERN stack**.

Trip Pilot AI allows users to upload travel booking documents such as **flight tickets, hotel bookings, and travel PDFs/images**, automatically extract travel information using **AI**, and generate a structured **travel itinerary**.

Built as part of the **Orbitra Technologies – Junior Full Stack Developer (MERN + AI) Assignment**.

## 📦 Installation

### Clone Repository

```bash
git clone YOUR_REPOSITORY_URL
cd trip-pilot-ai
```

### Frontend Setup

```bash
cd frontend
npm install
npm run dev
```

### Backend Setup

```bash
cd backend
npm install
npm run dev
```
